# Agent Skills

**Agent Skills** is a **collection of production-ready agent skills** designed for **Codex** and **Claude**.
All skills follow a **strict structure**, **technical consistency**, and produce **standards-aligned outputs**, enabling reliable automation and seamless integration with multiple environments.

All skills are **generated using a dedicated GPT** specialized in skill creation:
[Agent Skill Creator GPT](https://chatgpt.com/g/g-69c101c983688191ae2186efcd766a33-agent-skill-creator)

---

## 🧩 Features

* **Production-ready structure**: Each skill adheres to strict formatting and technical standards.

* **Consistency across skills**: Ensures all outputs are uniform and predictable.

* **Standards-aligned outputs**: Skills generate responses that follow predefined rules and best practices.

* **Extensible design**: Skills can be combined or extended for complex workflows.

* **Automated skill creation**: Built using a GPT specialized in generating high-quality agent skills.

* **Integration-ready**: Designed for seamless use with Codex and Claude-based agents.

---

## 🚀 Usage

1. **Clone the repository**

   ```bash
   git clone https://github.com/rwxce/agent-skills.git
   cd agent-skills
   ```

2. **Generate or add skills**

   Use the **Agent Skill Creator GPT** to create new skills or improve existing ones.

3. **Integrate with agents**

   Import the generated skills into your **Codex** or **Claude** agent environment.
   Ensure all dependencies and runtime requirements for your agent platform are met.

4. **Test and validate**

   Each skill should be tested individually and in combination to ensure outputs are consistent and standards-aligned.
